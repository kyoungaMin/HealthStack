## 평가 기준

### 1. 검색 품질 (Retrieval Quality)
- [ ] 질문 키워드 → 관련 증상 매핑 정확도
- [ ] 복합 조건 처리 능력 (증상 + 시간대 + 제외 조건)
- [ ] 동의어/유사어 인식 (더부룩 ↔ 소화불량)

### 2. 추천 품질 (Recommendation Quality)
- [ ] direction 구분 정확도 (recommend vs avoid)
- [ ] priority 기반 정렬
- [ ] rationale_ko 품질 (사용자 친화적 설명)

### 3. 레시피 추천 품질
- [ ] 증상-레시피 연관성
- [ ] meal_slot / best_time 적절성
- [ ] 조리 난이도 고려

### 4. 고급 기능 평가
- [ ] 약물 상호작용 안내 (D6)
- [ ] 영양소 → 식재료 역검색 (D1, B6)
- [ ] 조리법 포함 답변 (A10)
